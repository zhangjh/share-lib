A private common share libs developed for Java apps.

## Usage
[查看最新版本](https://central.sonatype.com/artifact/me.zhangjh/share/1.0.3)

```xml
    <dependency>
        <groupId>me.zhangjh</groupId>
        <artifactId>share</artifactId>
        <version>${version}</version>
    </dependency>
```
